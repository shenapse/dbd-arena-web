These policies apply to all players across all regulations. They set out
expected behaviour, how edge cases are handled, and how data is managed.

## Policies

- **[Participation and Conduct](/handbook/policies/participation-and-conduct/)**
  — Standards for sportsmanship, communication, and fair play during matches
  and in the Discord server.

- **[Privacy and Public Data](/handbook/policies/privacy-and-public-data/)** —
  What player data is stored, what is publicly visible, and how to request
  removal.
