## Game Structure

This fixed-role 1v1 format assigns one player to the killer side and the other
to the survivor side for the entire series. Unlike the symmetric regulation,
players do not alternate roles between rounds. They compete in a best-of-X
(BoX) series, provisionally Bo3 or Bo5.

Each round records one chase time and compares it with a threshold configured
for the selected killer. The killer player takes the round when the chase time
is below the threshold; the survivor player takes it when the chase time meets
or exceeds the threshold. These per-killer thresholds are upcoming balancing
values and are not yet final. For example, if The Mastermind (Wesker) has a
100-second threshold, the killer wins below 100 seconds and the survivor wins
at 100 seconds or longer.

| Attribute | Value |
|---|---|
| Team structure | 1 × Killer vs 1 × Survivor |
| Map selection | Provisional — to be confirmed before launch. |
| Build restriction | Provisional — a restricted killer pool is planned. Rules not yet final. |
| Pre-match killer reveal | Yes — the killer character is known to both sides before the match starts. |
| Pre-match player reveal | Yes |
| Number of rounds | Provisional — 3 or 5 |
