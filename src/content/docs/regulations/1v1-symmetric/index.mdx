## Game Structure

The classical 1v1 format where each player takes both the killer and survivor side,
competing in best-of-3. In each round, both
players record a chase time, and the player with the longer chase time takes the
round.

| Attribute | Value |
|---|---|
| Team structure | 1 × Player A vs 1 × Player B |
| Map selection | Fixed per killer character |
| Build restriction | Yes — see per-killer balancing |
| Pre-match killer reveal | Yes — the killer character is known to both sides before the match starts. |
| Pre-match player reveal | Yes |
| Number of rounds | 3 (Bo3) |
