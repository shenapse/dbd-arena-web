## Game Structure

A symmetric duel where each player takes both the killer and survivor side,
competing on chase time. Announced for a future season.

| Attribute | Value |
|---|---|
| Team structure | 1 × Player A vs 1 × Player B |
| Map selection | Provisional — to be confirmed before launch. |
| Build restriction | Provisional — a restricted killer pool is planned. Rules not yet final. |
| Rated | Yes |

## Queue requirement

**Eligibility:** Planned for registered ranked players. Details to be confirmed.

**Queue mode:** Ranked (planned)

Role is decided by how you queue — you do not declare it separately.

| Who | How they queue | Declares at queue time |
|---|---|---|
| Each player | Queues solo. (Provisional.) | One killer character from a restricted pool. (Provisional.) |

## Result facts

Players report these facts; the opponent approves the outcome.

| Fact | Field | Notes |
|---|---|---|
| Winning player | `winning_team` | Which player won. Omitted on a draw. (Provisional.) |
| Draw | `draw` | Marks a drawn duel. (Provisional.) |
| Chase time | `chase_time_seconds` | Seconds survived while on the survivor side. (Provisional.) |
