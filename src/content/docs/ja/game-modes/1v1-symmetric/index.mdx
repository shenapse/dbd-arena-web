## ゲーム構成

各プレイヤーがキラー側とサバイバー側の両方を担当し、チェイス時間を競う典型的な1対1形式。
3ラウンド制で行い、各ラウンドでは両プレイヤーのチェイス時間を記録し、より長いチェイス時間を記録したプレイヤーがそのラウンドを獲得する。2ラウンドを先取したプレイヤーが勝者。

| 項目 | 内容 |
|---|---|
| チーム構成 | プレイヤーA 1人 vs プレイヤーB 1人 |
| マップ選択 | キラーごとに固定 |
| 構成制限 | あり — キラー別バランス調整を参照 |
| 試合前のキラー公開 | あり |
| 試合前のプレイヤー公開 | あり |
| ラウンド数 | 3 (Bo3) |
