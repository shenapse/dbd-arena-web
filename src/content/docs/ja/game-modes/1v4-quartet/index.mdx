## キュー参加条件

サバイバーは4人のフルパーティでキューに参加する（=マッチメイキングリクエストを行う）。

| 対象 | キューへの参加方法 | キュー参加時の申告内容 |
|---|---|---|
| キラープレイヤー | 1人でキューに参加 | 使用するキラー1体 |
| サバイバープレイヤー | 4人グループでキューに参加 | なし |

## レーティング

1v4 Quartet Ranked では、ゲームモードごと、かつキラーごとに分かれたレーティングを使用する。

このレーティングの範囲は、次の両方のフェーズで同じである:

- マッチメイキングフェーズ: 該当するキラーの1v4 Quartet Ranked 用レーティングを使ってマッチを組む
- レーティング更新フェーズ: 試合後に、同じ範囲のレーティングを更新する

1試合で使用・更新されるレーティングは次の通り:

- キラープレイヤー: 使用したキラーのレーティング
- サバイバーパーティ: 対戦したキラーに対する**サバイバーグループ**のレーティング

例として、プレイヤーKがナースを使用し、サバイバーチームS=(s1, s2, s3, s4)と対戦した場合、
使用・更新されるレーティングは次の通り:

- (K, \{1v4 Quartet・ランク ゲームモード / ロール=キラー / キラー=ナース\})
- (S, \{1v4 Quartet・ランク ゲームモード / ロール=サバイバー / キラー=ナース\})
- **対象外**: (**s1**, \{1v4 Quartet・ランク ゲームモード / ロール=サバイバー / キラー=ナース\})
- **対象外**: (S, \{**別のゲームモード** / ロール=サバイバー / キラー=ナース\})
- **対象外**: (S, \{別のゲームモード / ロール=サバイバー / キラー=**ブライト**\})
