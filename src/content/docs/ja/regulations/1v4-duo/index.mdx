## キュー参加条件

サバイバーはデュオとしてキューに参加する必要があります。

| 対象 | キューへの参加方法 | キュー参加時の申告内容 |
|---|---|---|
| キラープレイヤー | 1人でキューに参加 | 使用するキラー1体 |
| サバイバーパーティ | デュオ (2人1組のグループ) でキューに参加 | なし |

## リザルト情報

| 項目 | 説明 |
|---|---|
| 勝利陣営 | キラーまたはサバイバーのどちらが勝利したか。引き分けの場合は記録しません |
| 引き分け | 試合が引き分けであることを示します。勝利陣営と同時に記録することはできません |
| 残り発電機数 | 試合終了時に修理されていなかった発電機の数（0〜5） |
| フック数 | 試合中にキラーが行ったフックの合計数（0〜12） |
| キル数 | 死亡したサバイバーの数（0〜4） |

## レーティング

このレギュレーションでは、キラーごとに厳密に分かれたレーティングを使用します。
詳しくは [1v4 Quartet Ranked](/ja/regulations/1v4-quartet/) のレーティング項目を参照.
